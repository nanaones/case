
## REST API 기능 

### 회사명 자동완성
회사명의 일부만 들어가도 검색됩니다.  
### 태그명으로 회사 검색
관련된 회사가 검색됩니다.  
다국어로 검색이 가능합니다.  
일본어 태그로 검색을 해도 한국 회사가 노출이 됩니다.  
タグ_4로 검색 했을 때, 원티드랩 회사 정보가 노출됩니다.  
동일한 회사는 한번만 노출됩니다.
